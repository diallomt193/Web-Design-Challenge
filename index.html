Summary: Latitude vs. X

The purpose of this project was to analyze how weather across 500+ cities around the world changes as you get closer to the equator. To accomplish this analysis, we used OpenWeatherMap API to assemble the dataset in order to create a representative model of weather across world cities.


Once the dataset was assembled, Matplotlib was used to build a series of plots to showcase the relationships between several variables of weather vs. latitude. Some of the factors we looked at in this case include temperature, cloudiness, humidity and wind speed. To showcase the analysis completed, we provided data and visualizations as part of the analysis, including explanations and descriptions of trends and correlations.
